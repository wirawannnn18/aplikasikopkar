## Layout Customization

### Grid System
Dashboard menggunakan 12-column responsive grid system:
- **Drag & Drop**: Pindahkan widget dengan drag & drop
- **Resize**: Drag corner widget untuk resize
- **Snap to Grid**: Widget otomatis snap ke grid
- **Responsive**: Layout otomatis adjust untuk mobile

### Layout Templates
**Predefined Templates**:
- **Executive**: Overview KPI untuk management
- **Operational**: Detail operasional untuk staff
- **Financial**: Focus pada metrik keuangan
- **Analytics**: Advanced analytics dan trends
- **Custom**: Layout yang dibuat sendiri

### Creating Custom Layout
1. Click "Customize Layout" button
2. Drag widgets dari sidebar ke main area
3. Resize dan position widgets sesuai kebutuhan
4. Save layout dengan nama custom
5. Set sebagai default layout (optional)

## Widget Configuration

### Adding Widgets
1. Click "+" button atau "Add Widget"
2. Select widget type dari gallery
3. Configure data source dan parameters
4. Set display options dan formatting
5. Position widget di dashboard

### Widget Types Available
- **KPI Widgets**: Single metrics dengan trend indicators
- **Chart Widgets**: Line, bar, pie, doughnut, radar charts
- **Table Widgets**: Tabular data dengan sorting/filtering
- **Gauge Widgets**: Progress indicators dan meters
- **Heatmap Widgets**: Activity patterns visualization

### Widget Settings
**Data Configuration**:
- Data Source: Select dari available sources
- Time Range: Set periode data (last 7 days, 30 days, etc.)
- Filters: Apply category atau member filters
- Aggregation: Sum, average, count, min, max

**Display Options**:
- Title: Custom widget title
- Colors: Color scheme dan branding
- Format: Number format, currency, percentages
- Size: Widget dimensions (1x1, 2x1, 2x2, etc.)

## Role-based Customization

### Role Hierarchy
1. **Viewer**: Read-only access, basic widgets
2. **Staff**: Standard access, operational widgets
3. **Supervisor**: Enhanced access, team analytics
4. **Manager**: Management access, strategic KPIs
5. **Admin**: Full access, system configuration
6. **Super Admin**: Complete system control

### Permission-based Features
**Widget Visibility**: Different widgets untuk different roles
**Data Access**: Role-based data filtering
**Export Permissions**: Control export capabilities
**Customization Rights**: Who can modify layouts

### Creating Role Templates
1. Login sebagai Admin/Super Admin
2. Go to Settings > Role Management
3. Select role untuk customize
4. Configure available widgets dan permissions
5. Set default layout untuk role
6. Save template untuk future users

### Sharing Configurations
- **Export Config**: Save configuration sebagai file
- **Import Config**: Load configuration dari file
- **Share Dashboard**: Share specific dashboard dengan users
- **Template Library**: Organization-wide template sharing
