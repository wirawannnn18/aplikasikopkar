Panduan Debug

Ikuti langkah-langkah berikut untuk debug masalah:

  1. Buka aplikasi di http://localhost:3000
  2. Login sebagai Super Admin (superadmin / super123)
  3. Buka Developer Console (F12)
  4. Klik menu "Pengaturan Sistem"
  5. Lihat apakah ada error di Console
Checklist Debugging
Periksa hal-hal berikut:
Kemungkinan Masalah & Solusi

Penyebab:
  • Tidak login sebagai Super Admin
  • Cache browser lama
Solusi:
  1. Pastikan login dengan username: superadmin
  2. Refresh halaman (Ctrl+F5)
  3. Clear cache browser
  4. Logout dan login kembali

Penyebab:
  • JavaScript error
  • Fungsi tidak terdefinisi
Solusi:
  1. Buka Developer Console (F12)
  2. Lihat tab Console untuk error
  3. Screenshot error dan laporkan
  4. Refresh halaman (Ctrl+F5)

Penyebab:
  • Event listener tidak terpasang
  • JavaScript error
Solusi:
  1. Buka Console (F12)
  2. Klik tombol dan lihat error
  3. Refresh halaman
  4. Coba akses menu Backup & Restore langsung dari sidebar

Penyebab:
  • Bukan Super Admin
  • Session expired
Solusi:
  1. Logout
  2. Login kembali sebagai superadmin
  3. Cek role di Console: JSON.parse(localStorage.getItem('currentUser')).role
  4. Harus menampilkan: "super_admin"
