Fix Auth Migration - Supabase MCP

Masalah yang Ditemukan:
  1. supabase.from is not a function - Supabase client belum diinisialisasi
  2. Cannot read properties of undefined (reading 'admin') - Admin API tidak tersedia di client-side
Solusi:

Kita akan memperbaiki file js/authMigration.js dengan:

  1. Menginisialisasi Supabase client dengan benar
  2. Menggunakan direct database operations untuk membuat user
  3. Menghapus penggunaan admin API yang tidak tersedia
